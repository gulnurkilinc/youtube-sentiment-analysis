### 🦋 Requirements

- 🐍 Python 3.10+
- 💚 Node.js 18+
- 🔑 YouTube Data API v3 key
- 🔑 OpenAI API key

### 🌸 1. Clone the Repo
```bash
git clone https://github.com/gulnurkilinc/youtube-sentiment-analysis.git
cd youtube-sentiment-analysis
```

### 🌸 2. Setup Backend
```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
# Open .env and add your API keys
```

### 🌸 3. Setup Frontend
```bash
cd frontend
npm install
```
